## âš™ï¸ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/Amit-Thakor/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Open in browser**
   Visit <http://localhost:3000> to view the site.
